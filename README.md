# Estar Seguro Polymer Pvt. Ltd.

Market-presence website for Estar Seguro Polymer Pvt. Ltd., built as a static
Next.js site and deployed to GitHub Pages.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validation

```bash
npm test
npm test
```

The second command validates the same subpath configuration used by GitHub
Pages. Pushing to `main` triggers the deployment workflow.
